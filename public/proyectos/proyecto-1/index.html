    caracteristicas

    'Proyecto 1' es un proyecto de ejemplo inexistente, creado para ilustrar la estructura y el formato de un portafolio profesional.

    Posee 4 secciones principales: Características, Imágenes, Habilidades y Tecnologías.
    Cada sección está diseñada para proporcionar una visión completa del proyecto; los titulos de las se pueden cambiar, modificando los valores de las variables en el bloque 'summary' al inicio de este archivo, cada 'link', representa una sección y tiene dos propiedades: 'href' y 'text'.
    - 'href' define el identificador de la sección, que se utiliza para la navegación interna.
    - 'text' define el título visible de la sección.
